# 🚀 RAG-FLINK-KAFKA

A **Real-time RAG-powered Financial Chatbot** built with **Apache Flink + Kafka + FastAPI + Next.js** ⚡  
It streams user queries in real-time, retrieves relevant financial data, and generates **AI-powered insights**.
